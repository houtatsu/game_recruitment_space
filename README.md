# ゲームコミュ

## サイト概要
### サイトテーマ
- 一緒にゲームする人を募集したり、募集している人を探したり出来るコミュニティーサイト


### テーマを選んだ理由
- もともとゲームが好きで、プレイスタイルや時間を決めて募集や探すことができるサービスがあれば便利だと考えこのテーマにしました。


### ターゲットユーザ
- ゲームが好きな人
- ゲームをやっている人
- 一緒にゲームをやる人が欲しい人

### 主な利用シーン
- 募集している人を探す時
- 自分が募集する時
